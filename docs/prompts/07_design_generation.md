# Prompt 1

Toma el contexto de lo que sabes del proyecto usando @planning.md, @tasks.md
y los archivos en @docs/adrs para generar un prompt que pueda llevar
a v0 para que me genere 2 propuestas de diseño de la UI del dashboard.

El prompt debe ser lo más detallado posible y debe incluir:

- El contexto del proyecto.
- Las tareas pendientes.
- Los ADRs relevantes.
- Las especificaciones de la UI.
- Las especificaciones de la funcionalidad.

Solo quiero el diseño de la UI, no una propuesta de implementación.

Genera el prompt en markdown, con un máximo de 1000 caracteres.
