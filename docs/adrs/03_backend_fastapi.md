# 3. Backend con FastAPI

Fecha: Julio 3 de 2025

## Status

Aceptada.

## Contexto

Se requiere un framework backend en Python que permita construir APIs REST de forma rápida, con validación automática, documentación integrada y alto rendimiento.

## Decisión

Se selecciona FastAPI como framework para el desarrollo del backend.

## Consecuencias

Facilita el desarrollo ágil de endpoints, la validación de datos y la integración con herramientas modernas de Python. Proporciona documentación automática y buen rendimiento.
