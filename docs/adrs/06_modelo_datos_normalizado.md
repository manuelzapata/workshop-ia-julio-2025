# 6. Modelo de datos normalizado

Fecha: Julio 3 de 2025

## Status

Aceptada.

## Contexto

El sistema debe almacenar información compleja y relacionada proveniente de un dataset externo. Es necesario garantizar la integridad, evitar redundancias y facilitar consultas eficientes.

## Decisión

Se adopta un modelo de datos normalizado en la base de datos, estructurando la información en tablas relacionadas según las mejores prácticas de diseño relacional.

## Consecuencias

Mejora la integridad de los datos, reduce la redundancia y facilita el mantenimiento y la evolución del modelo de datos a futuro.
