# 2. Frontend con Next.js

Fecha: Julio 3 de 2025

## Status

Aceptada.

## Contexto

Se requiere un framework moderno y eficiente para construir interfaces de usuario reactivas y escalables, con soporte para renderizado del lado del servidor y despliegue sencillo en la nube.

## Decisión

Se selecciona Next.js como framework para el desarrollo del frontend del dashboard.

## Consecuencias

Permite aprovechar el ecosistema de React, facilita el SSR (Server Side Rendering), la generación de sitios estáticos y el despliegue en plataformas como Vercel.
